# Referencias academicas

As referencias abaixo podem ser usadas no relatorio e nos slides. Caso o professor exija um padrao especifico, como ABNT, IEEE ou APA, ajuste pontuacao, italico e datas de acesso conforme as regras da disciplina.

## Referencias sugeridas

1. KNUTH, Donald E.; MORRIS, James H.; PRATT, Vaughan R. Fast pattern matching in strings. *SIAM Journal on Computing*, v. 6, n. 2, p. 323-350, 1977.

2. CORMEN, Thomas H.; LEISERSON, Charles E.; RIVEST, Ronald L.; STEIN, Clifford. *Introduction to Algorithms*. 4. ed. Cambridge: MIT Press, 2022.

3. SEDGEWICK, Robert; WAYNE, Kevin. *Algorithms*. 4. ed. Boston: Addison-Wesley, 2011.

4. GUSFIELD, Dan. *Algorithms on Strings, Trees, and Sequences: Computer Science and Computational Biology*. Cambridge: Cambridge University Press, 1997.

5. AHO, Alfred V.; HOPCROFT, John E.; ULLMAN, Jeffrey D. *The Design and Analysis of Computer Algorithms*. Boston: Addison-Wesley, 1974.

6. PYTHON SOFTWARE FOUNDATION. *time - Time access and conversions*. Python Documentation. Disponivel em: https://docs.python.org/3/library/time.html. Acesso em: 16 jun. 2026.

7. ORACLE. *Class System*. Java Platform Documentation. Disponivel em: https://docs.oracle.com/en/java/javase/21/docs/api/java.base/java/lang/System.html. Acesso em: 16 jun. 2026.
